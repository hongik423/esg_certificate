## 🚀 **빠른 시작 가이드**

### 1️⃣ **저장소 클론**
```bash
git clone https://github.com/hongik423/m_center_landingpage.git
cd m_center_landingpage
```

### 2️⃣ **의존성 설치**
```bash
npm install
```

### 3️⃣ **환경변수 설정 (별-AI상담사 활성화)**
```bash
# .env.local 파일 생성
cp .env.local.example .env.local

# Gemini API 키 설정 (필수!)
# https://aistudio.google.com/에서 발급
GEMINI_API_KEY=AIzaSy...
```

### 4️⃣ **개발 서버 실행**
```bash
npm run dev
# http://localhost:3000 접속
```

### 5️⃣ **AI 상담사 테스트**
```bash
# 우하단 별-AI상담사 아이콘 클릭
# "안녕하세요" 메시지 전송으로 테스트
```
